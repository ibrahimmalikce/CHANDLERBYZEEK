CHANDLER x ZEEK Street Style Collection
Project Overview
This project is a high-end, responsive landing page for CHANDLER by ZEEK, a "Captivating Cities" shirt collection built for street style and speed enthusiasts. The website features a dark, modern aesthetic with purple accents, heavy typography, and interactive elements designed to showcase the brand's unique vibe.
Core Technologies
HTML5: Structured semantically for better accessibility.  

CSS3:

CSS Grid & Flexbox: Used for advanced layouts like the split-screen "Roots" section and the product catalog.  

Custom Properties (Variables): Centralized color scheme (--purple, --black, etc.) for easy theme management.  

Media Queries: Sophisticated breakpoints ensure the site adapts from desktop widths down to 480px smartphones.  

Vanilla JavaScript: Handles the logic for opening/closing the ordering modal and displaying success "toast" notifications after a purchase request.
